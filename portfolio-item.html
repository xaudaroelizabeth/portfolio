Twimbaweb app

HTML, CSS, & JavaScript.

GitHub Repo Live version

Twimba is a Twitter-inspired social media app built using vanilla HTML, CSS, and JavaScript. It features a dynamic feed where users can create tweets, like posts, retweet content, and add comments in real time.

The project uses localStorage to persist data in the browser, ensuring tweets remain even after refreshing the page. It demonstrates core frontend concepts such as DOM manipulation, event handling, and state management without any frameworks.

Twimba is designed as part of a frontend development learning project, focusing on building interactive user interfaces using only vanilla web technologies.

This project helped strengthen understanding of how data flows through a UI, how events update application state, and how to render dynamic content efficiently in the browser.
